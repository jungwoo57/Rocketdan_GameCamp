로켓단 게임캠프 9기 과제 제출용 프로젝트입니다.

과제에 포함된 기능 위주로 구현하였습니다

적은 지정된 경로로 움직이며 특정 구간이 지나면 광폭화하여 속도가 빨라집니다.

플레이어는 A,D 혹은 방향키 <- 와->로 좌우로 움직일 수 있습니다.

공격은 자동으로 진행하며 움직일때는 정방향 위로 움직이지 않을 경우 가장 가까운 적을 타겟하여 공격합니다.

적은 여러개의 몸통과 하나의 머리로 이루어져있고 몸통에 데미지를 가하면 몸통의 HP가 줄어듭니다.

몸통은 각각 HP를 가지고 있고 HP가 0이 될 시 해당 몸통이 처치됩니다.

모든 몸통을 처치하면 게임을 승리합니다.

만약 적을 처치하지 못하고 기준선에 적이 도착하게되면 게임을 패배합니다.
